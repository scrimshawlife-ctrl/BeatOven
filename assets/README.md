# BeatOven Assets

This directory contains visual assets for the BeatOven repository.

## Hero Header Image

**Required:** `hero-header.jpg`

Please save the vintage-style hero header image to this directory as `hero-header.jpg`.

The image should be:
- Filename: `hero-header.jpg`
- Location: `/assets/hero-header.jpg`
- This image will be displayed at the top of the main README.md

## Adding the Image

Save your hero header image to this directory with the filename `hero-header.jpg`.
